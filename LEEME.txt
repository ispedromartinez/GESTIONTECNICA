  INFORME MANTENIMIENTO CLIMA v3.0
═══════════════════════════════════════════════

ARCHIVOS (deben estar todos en la misma carpeta):
  server.js               → Servidor Node.js
  informe_clima_app.html  → Aplicación web
  package.json            → Dependencias
  logo.jpeg               → Logo Centrales Clima ← NUEVO

INSTALACIÓN (solo primera vez):
  1. Instalar Node.js: https://nodejs.org (versión LTS)
  2. Abrir terminal en esta carpeta
  3. Ejecutar: npm install

USAR CADA DÍA:
  1. Abrir terminal en la carpeta
  2. Ejecutar: node server.js
  3. Abrir informe_clima_app.html en el navegador

IMPORTANTE: el archivo logo.jpeg debe estar
en la misma carpeta que server.js
